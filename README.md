# 🎊 年会抽奖系统

一个基于 React + TypeScript + Vite 开发的年会抽奖程序，支持滚动姓名抽奖、多奖项管理、烟花特效等功能。

## ✨ 功能特性

- 🎯 **滚动抽奖** - 点击开始后姓名快速滚动，点击停止后减速选中中奖者
- 🏆 **多奖项管理** - 支持特等奖、一等奖、二等奖、三等奖等多个奖项等级
- 🎨 **喜庆风格** - 红金配色，适合年会节日氛围
- 🎉 **烟花特效** - 中奖时触发精美的烟花动画
- 👥 **名单管理** - 支持手动输入、导入/导出参与者名单
- 💾 **数据持久化** - 使用 localStorage 保存数据，刷新不丢失
- 📊 **中奖记录** - 实时显示已中奖名单
- 🔄 **重置功能** - 支持清空当前奖项或全部重置

## 📖 使用说明

### 1. 添加参与者

点击左侧"参与者名单"区域，在弹出的文本框中输入姓名，每行一个。也可以导入 txt 或 csv 文件。

示例：
```
张三
李四
王五
赵六
```

### 2. 选择奖项

点击"奖项选择"区域的奖项按钮切换当前抽奖奖项。

奖项进度条显示当前奖项的中奖进度。

### 3. 开始抽奖

1. 确认参与者名单已添加
2. 选择要抽取的奖项
3. 点击"▶ 开始抽奖"按钮
4. 姓名开始快速滚动
5. 点击"⏹ 停止"按钮
6. 姓名减速后停在中奖者
7. 触发烟花特效，中奖者显示在主屏幕

### 4. 查看中奖记录

右侧"中奖记录"区域实时显示所有奖项的中奖名单。

### 5. 重置数据

- 右下角重置按钮：重置所有数据（参与者、中奖记录）
- 奖项选择器右侧"清空当前中奖记录"：仅清空当前奖项的中奖者

## 🎮 技术栈

- **框架**: React 18 + TypeScript
- **构建工具**: Vite 5
- **样式**: Tailwind CSS 3
- **图标**: Lucide React
- **特效**: canvas-confetti

## 🎨 配置奖项

默认奖项配置（可在 `src/types/index.ts` 中修改）：

- 特等奖: 1 名
- 一等奖: 3 名
- 二等奖: 5 名
- 三等奖: 10 名

## 🌟 部署

构建后的文件在 `dist/` 目录，可直接部署到任何静态文件服务器：

```bash
npm run build
```

然后将 `dist/` 目录的内容上传到服务器即可。

## 📝 注意事项

- 抽奖算法使用 Fisher-Yates shuffle 保证公平性
- 已中奖者会自动从候选池中移除，不会重复中奖
- 所有数据保存在浏览器 localStorage 中，清除浏览器数据会丢失
- 建议定期导出参与者名单作为备份
